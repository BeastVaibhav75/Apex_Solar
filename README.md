# Apex Solar

Under-construction landing page for Apex Solar Infra Pvt Ltd — solar panel installation and renewable energy services.

## Stack

- [Next.js](https://nextjs.org) (App Router)
- Tailwind CSS
- Framer Motion

## Development

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Deploy

Deploy on [Vercel](https://vercel.com) — import this repo and deploy with default Next.js settings.
